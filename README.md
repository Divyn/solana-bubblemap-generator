# Solana Historical USD Bubblemap Without Running a Node

## Build Bubblemap
```
node scripts/build-bubblemap.js <currency_address> <receiver_address> [since_date]
```

Example:
```
node scripts/build-bubblemap.js Es9vMFrzaCERmJfrF4H2FYD4KCoNkY11McCe8BenwNYB CapuXNQoDviLvU1PxFiizLgPNQCxrsag1uMeyk6zLVps 2025-09-24
```

## Serve HTML
npm run serve
